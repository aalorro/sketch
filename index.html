Load Image/s

Select one or more image files (JPG, PNG, etc.) to process. You can upload multiple images and use the Prev/Next buttons to browse through them individually before batch processing.

Tip: Use high-quality source images for best results. Larger images may take longer to process.

Skip Hatching

When checked, removes decorative hatching and pattern effects, giving you clean line-based sketches with crisp edges only.

Best for: Simple outlines, contour drawings, and minimalist styles.

When unchecked, hatching and brush textures are applied based on your chosen brush type.

Colorize

When checked, adds color to your sketch based on the original image, creating a colored sketch effect.

Combine with Saturation and Hue Shift sliders to fine-tune the color intensity and tone.

Tip: Works best with styles like Comic/Manga, Fashion Sketch, or Watercolor to create stylized colored artwork.

Prompt

An optional text description sent to external ML services or custom servers to guide image processing.

When it works: Only used if "Use external ML service" or "Use server style-transfer" is enabled.

Example prompts:

  • "dramatic cross-hatching with bold lines"
  • "soft watercolor wash effect"
  • "detailed pen stippling"

Note: Has no effect during default client-side processing.

Seed

A number that makes random effects (like scribbling or stippling) reproducible. Use the same seed to generate identical sketches from the same image.

Example: If Seed = 42 creates a style you like, you can reuse Seed 42 with other images for consistent results.

Leave blank or 0 for random variation each time.

Output Filename

Specify a custom prefix for downloaded sketch files. When batch processing, this prefix is applied to all generated files.

Default format: If left blank, files are named sketchify_YYYYMMdd_HHmmss.png (e.g., sketchify_20260220_143045.png)

Custom example: If you enter "my_artwork", files will be named my_artwork_1.png, my_artwork_2.png, etc.

Presets

Save and load your favorite configuration combinations (style, brush, intensity, saturation, etc.) for quick reuse.

How to save:

  1. Adjust all settings to your liking
  2. Enter a preset name (e.g., "dramatic-charcoal")
  3. Click "Save Preset"

How to load:

  1. Select a preset from the dropdown
  2. Click "Load Preset"

Presets are stored in your browser and persist across sessions.

Use GPU (WebGL)

Enables GPU-accelerated edge detection using WebGL for faster real-time previews on large images.

Benefits: Significantly faster processing on devices with modern GPUs.

Fallback: If GPU is unavailable or unsupported, processing automatically falls back to CPU.

Note: This is experimental and may have different results on some browsers.

Use External ML Service

Sends images to a custom ML endpoint (e.g., style transfer API) instead of using built-in sketch styles.

When to use: You have a custom ML model or proprietary style transfer service.

Privacy note: Images will be uploaded to your specified ML service ‚Äî use only with trusted endpoints.

Configuration:

  1. Enable this checkbox
  2. Enter your ML Service URL
  3. Upload images and click Generate

The Prompt field will be sent to your ML service if configured to use it.

Use Server Style-Transfer

Sends images to your own custom Flask/Python server for processing instead of using built-in sketch styles.

When to use: You have a local or remote server running (e.g., with OpenCV, advanced algorithms).

Setup:

  1. Run python server.py or python server_advanced.py
  2. Note the server URL (default: http://localhost:5000/api/style-transfer)
  3. Enable this checkbox and enter your Server API URL
  4. Upload images and click Generate

Note: Server must be running for this to work. Images stay on your server unless you configure otherwise.

Generate

Processes all uploaded images with your current settings (style, brush, intensity, etc.).

Processing:

  • Single image: Processes instantly in your browser
  • Multiple images: Processed sequentially to save memory
  • Progress bar shows real-time status

After generation: Download individual images as PNG/JPG or bundle all into a ZIP file.

Tip: For faster processing, reduce resolution or enable GPU acceleration.

Images are downloaded to your device ‚Äî nothing is stored on servers by default.

Instructions & Guide

Getting Started

  1. Upload Images: Click "Load image" and select one or more image files (JPG, PNG, etc.)
  2. Preview Settings: The preview updates in real-time as you adjust controls
  3. Generate Sketches: Click the Generate button to process your images
  4. Download Results: Download as PNG, JPG, or ZIP

Key Features

  • 20+ Sketch Styles: Contour, Hatching, Charcoal, Comic, Ink Wash, and more
  • Medium Control: Choose Pencil, Ink, Marker, or Pen to affect stroke appearance
  • Brush Effects: Add texture with Line, Hatch, Cross-hatch, or Charcoal effects
  • Real-time Preview: See changes instantly as you adjust intensity and stroke
  • Batch Processing: Upload multiple images and process them all at once
  • Multi-image Navigation: Use Prev/Next buttons to adjust settings per image before generating

Controls Explained

  • Intensity (1-10): Controls edge detection strength and detail level
  • Stroke (1-10): Adjusts line width and pattern density
  • Colorize: Add color tints to your sketches
  • Skip Hatching: Remove decorative patterns for clean lines
  • Resolution: Higher resolution = more detail but slower processing
  • Output Name: Custom prefix for downloaded files

Processing & Export

  • PNG Download: Lossless format, best for line-based sketches
  • JPG Download: Compressed format, smaller file size
  • ZIP Download: Bundle all processed images into one file (requires batch generation)
  • Custom Filename: Leave blank for auto-generated timestamp-based names

Tips & Tricks

  • Use high-quality source images for best results
  • Experiment with different mediums and brush types on the same image
  • Enable GPU acceleration for faster processing on large images
  • Save your favorite settings as presets for quick reuse
  • Use Undo/Redo (Ctrl+Z / Ctrl+Y) to experiment freely

Privacy Policy

Your Privacy Matters

Sketchify is committed to protecting your privacy. This privacy policy explains how we handle your data when you use our application.

What Data We Collect

  • Zero Image Storage: Your images are processed locally in your browser only. We never upload, store, or transmit your images to any server.
  • No Personal Information: We do not collect your name, email, location, or any personal identifying information for using Sketchify.
  • No Cookies or Tracking: Sketchify does not use cookies, analytics, or tracking pixels to monitor your activity.
  • Contact Form Data: If you submit the contact form, your message and email are sent to our server via Formspree and Discord webhooks for support purposes only.

How Your Data is Used

  • Local Processing: All image processing happens on your device using your browser's Canvas API. No data leaves your computer.
  • Contact Messages: Messages from the contact form are used only to respond to your inquiries and improve our service.
  • No Sharing: We never sell, trade, or share your data with third parties.
  • No Third-Party Analytics: We don't use Google Analytics or similar services that track user behavior.

External Services

  • PayPal: Donations are processed through PayPal. Their privacy policy applies to payment information.
  • Discord & Formspree: Contact form submissions are sent to these services for message delivery. Review their privacy policies for details.
  • GitHub/Hosting Provider: If you access Sketchify from a hosted domain, the hosting provider's privacy policy may apply to server logs.

Your Rights

  • No Login Required: Sketchify doesn't require an account, so no profile data is stored.
  • Local Storage: Your preset settings are stored only in your browser's localStorage and never leave your device.
  • Full Control: You can clear your browser data anytime to delete all stored settings.
  • No Profiling: We don't build profiles or use your activity for advertising or targeting.

Security

Sketchify is designed with privacy and security in mind:

  • All processing happens on your device (no network requests for image data)
  • HTTPS encryption for any data transmitted
  • Open-source code available for security review
  • No sensitive data stored on our servers

Changes to This Policy
